Mission Archive

열방과 이웃을 향한 동역

열민교회는 작은 교회이지만 국내와 해외의 선교지를 꾸준히 품고 기도하며 후원해왔습니다. 이곳에는 각 선교지의 시작, 동역자, 방문 기록, 현재 상황, 기도제목을 차근차근 기록합니다.

Overseas

해외 선교

먼 땅의 교회와 사역자를 기억하며, 기도와 후원과 방문으로 함께 걷습니다.

해외

몽골 선교

열민교회가 오랫동안 마음에 품어온 선교지입니다. 후원과 선교여행의 기록, 현지 사역자의 이야기를 한곳에 정리할 수 있습니다.

후원 시작
확인 후 입력
동역 선교사
확인 후 입력
방문 기록
선교여행 횟수와 주요 방문 연도 입력
현재 상황
현지 교회, 다음 세대, 지역 사역 현황 입력
기도제목
현지 사역자와 공동체를 위한 기도제목 입력
해외

태국 선교

복음이 필요한 지역과 사역자를 위해 기도하며 동역하는 해외 선교지입니다.

후원 시작
확인 후 입력
동역 선교사
확인 후 입력
주요 사역
교회 개척, 교육, 지역 섬김 등 확인 후 입력
현재 상황
현지 사역의 필요와 변화 입력
해외

베트남 선교

현지 사역의 문이 계속 열리도록 기도하며 함께 섬기는 선교지입니다.

후원 시작
확인 후 입력
동역 선교사
확인 후 입력
주요 사역
현지 교회, 교육, 돌봄 사역 등 확인 후 입력
기도제목
사역 지속과 현지 공동체를 위한 기도제목 입력
해외

일본 선교

가깝지만 복음의 필요가 큰 일본 땅을 위해 기도하고 동역합니다.

후원 시작
확인 후 입력
동역 선교사
확인 후 입력
현재 상황
지역 교회와 사역 환경 입력
기도제목
복음의 문과 사역자의 건강을 위한 기도제목 입력

Domestic

국내 선교

가까운 지역의 작은 교회와 마을을 기억하며, 같은 복음 안에서 함께 서갑니다.

국내

조도섬 선교

섬 지역의 교회와 이웃을 기억하며 기도와 방문, 필요한 섬김으로 동역합니다.

동역 시작
확인 후 입력
동역 교회/사역자
확인 후 입력
방문 기록
방문 연도와 섬김 내용 입력
현재 필요
지역 교회와 마을의 필요 입력
국내

금산 천을리 선교

농촌 지역의 교회와 마을을 품고, 지역 안에 복음의 위로가 흐르도록 동역합니다.

동역 시작
확인 후 입력
동역 교회/사역자
확인 후 입력
주요 섬김
예배, 방문, 마을 섬김 등 확인 후 입력
기도제목
지역 복음화와 마을 공동체를 위한 기도제목 입력
